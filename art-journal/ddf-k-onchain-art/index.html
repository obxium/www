In the original JSON is metadata.image_data, which contains the actual SVG code for the image!

So, if you save the contents of the image_data field as a file named something like DDF-K.svg, you can both view the contents with your own human eyeballs, and also invoke UNIX beard spells like file and shasum to even further validate the media file compact signature.

Validation Spells

Get media file magic.

file DDF-K.svg
DDF-K.svg: SVG Scalable Vector Graphics image

Get compact signature from SHA 256 hash of media file.

echo $(shasum -a 256 DDF-K.svg | awk '{print $1}') | \
    awk '{print substr($0, 1, 8) substr($0,length($0)-7, 8)}'
